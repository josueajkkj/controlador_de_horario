# control_horario
Proyecto del **Día 1** del plan de aprendizaje de **Antigravity** enfocado en fundamentos, Vibe Coding y ejercicios básicos de JavaScript.

---

## Objetivo del proyecto

Comprender los conceptos iniciales del sistema Antigravity:

- Qué es Vibe Coding
- Diferencia entre entorno web y entorno local
- Qué es modo Planning
- Qué es modo Fast
- Qué modelo usar para generar una app completa

Además, se desarrollan micro-ejercicios en JavaScript para practicar lógica básica y generar commits de progreso.
